Instructions

  1. This tool is used to build a Data Model for S3Model. The Data Model is a collection of components that define the data elements of a Concept. The components are listed in the scrollable menu on the left.
  2. Click on List button to see a list of the components of that type.
  3. Click on the Add button to add a new component of that type.
  4. Clusters are grouping components, like a section of a form.
  5. DataModels are the wrapper component that creates a package container. In addition to a *data* Cluster, it also carries all of the meta-data describing the use of the model.
  6. DataModels and Clusters are built from the bottom up. That is, you must first build the components that make up the Cluster, and then build the Cluster. You must then build the DataModel, and add the Cluster to the DataModel.
