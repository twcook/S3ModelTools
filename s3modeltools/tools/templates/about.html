About

Model Tools is a set of tools for working with models in the S3 Model. It is a Eclipse plug-in that provides a graphical editor for the S3 Model, and a set of wizards for creating and editing models.

The S3 Model is a model for describing the structure of a software system. It is based on the Unified Modeling Language (UML), but it is a subset of UML that is focused on the structure of a system. The S3 Model is used by the S3 Modeler to generate code for a system.

The S3 Model Tools are available under the Eclipse Public License.

For more information, see the S3 Model web site.
